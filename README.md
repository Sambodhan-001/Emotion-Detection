Libraries used in Project
OpenCV
OpenCV is an open-source library for the computer vision. It provides the facility to the machine to recognize the faces or objects.

NumPy
NumPy is a library for the Python programming language, adding support for large, multi-dimensional arrays and matrices, along with a large collection of high-level mathematical functions to operate on these arrays.

pandas
It is a software library written for the Python programming language for data manipulation and analysis.

Keras
It is an open-source software library that provides a Python interface for artificial neural networks.

matplotlib
It is a plotting library for the Python programming language and its numerical mathematics extension NumPy
